before use
(explanations are already in comments in first lines of the js package but they'll be more readable here)
This package presents 4 ways to hide, display and toggle elements with an attached trigger on your platform.
  • a button hidding an element or a set of elements
  • a button displaying an element or a set of elements
  • a button toggling an element or a set of elements (hides the element if displayed, else hide it)
  • a system of "tabs" : several of buttons, each one displaying a corresponding tab.

There's no skill in Javascript needed for you to use the features but it will be much easier if you're comfortable with HTML & CSS. You can use this package on your tumblr, as the administrator of your forum (NOT as a member since you won't be able to add JS) or in any HTML page actually, like this one for example.

Triggers and targets can be of any type (div, a, span, button, p, custom elements...) but the triggers need to be recognized with a class specific to the feature used. Moreover, they can be placed anywhere you want (sometimes within one another).
features & classes of triggers
noteWhen talking about a trigger, it may content the attributes target="something", toggling="something", children="something" where something is the tag name. You need to add . for a class or # for an id. For example :
(1,2,3) class="toggle/hide/display" The trigger hides/display/toggles the target.

(take either toggle, hide or display, not all of them at the same time)

avant utilisation
(explications fournies en commentaires au début du package js mais plus lisibles ici)
Le package présente 4 façons de cacher et afficher des éléments attachés à un trigger.
  • bouton cachant un ou plusieurs éléments
  • bouton affichant un ou plusieurs éléments
  • bouton cachant et affichant alternativement un ou plusieurs éléments
  • un système "d'onglets" : plusieurs boutons, chacun d'eux affichant un onglet correspondant.

Il ne faut pas avoir de connaissances en Javascript pour utiliser ce package mais il sera beaucoup plus facile de le manipuler en étant à l'aise avec le HTML & le CSS. Vous pouvez l'utiliser sur votre Tumblr, sur un forum dont vous êtes administrateurice (et PAS en tant que membres sachant que vous n'aurez pas l'autorisation d'ajouter du JS), ou sur n'importe quelle page HTML en fait, comme cette dernière par exemple.
